Rotary Advent Calendar

This project came about when the director of my department mentioned that it would be nice to have an advent-style calendar as a promotional/educational tool. I wanted my calendar to serve the same basic function as an advent calendar, but to be visually distinctive. I kicked around a few design concepts, but after coming across jQuery.path, I got the idea to model it after a rotary dial phone.

The calendar days form a ring around a central content area and rotate to the top when clicked, revealing the appropriate content once the rotation is complete. If an invalid day (i.e. > today's date) is clicked, the rotation is skipped and the content area is populated with a message informing the user when they'll be able to view the requested content.

Technologies Used:

  • jQuery
  • jQuery.path
  • CSS3

Features of Note:

  • jQuery.path animates selected day to top position around the central content area
  • Flexible enough to handle every month, takes leap year into account
  • Rotational direction depends on shortest path

ASK Night Participant List

Alumni Sharing Knowledge (ASK) Night is an annual networking event hosted by my employer in which current students have a chance to network with alumni volunteers. In order to help the students prepare for the event, my department posts a list of the attendees. The original list needed to be manually constructed using data submitted by the attendees, and could only contain limited information. This updated list pulls data and images directly from the attendee registration form, and allows students to easily filter and search. The list is designed to eliminate duplicate data entry, and to convey a wide variety of information about each attendee in a compact format.

Technologies Used:

  • jQuery
  • Wufoo
  • CSS3

Features of Note:

  • Participant information is gathered via a Wufoo form
  • List is populated by form data using Wufoo API
  • jQuery filter controls allow users to search based on keyword and grad school attendance

Career Fair Participant List

This project was an extension of the ASK Night Participant List, with the goal of creating a list of employers who will be attending my department's Career Fair event. The major difference with this version is the inclusion of greatly enhanced filter controls. This was made possible by the use of several multiple choice fields, the data from which lent itself to easy filtering of the participants.

Technologies Used:

  • jQuery
  • Wufoo
  • CSS3

Features of Note:

  • Based on the ASK Night Participant List design
  • Uses Wufoo to collect and populate participant data
  • Filter controls greatly enhanced due to improved form design
